Comandos de Colores Usando CSS

1. color

Especifica el color del texto

background-color

Especifica el color del fondo del elemento. También puedes usar background para combinar propiedades. Ver más ejemplos aquí.

border-color

Especifica el color del borde. Debes también definir un border-style.

box-shadow

Le añade una sombra al contenedor con el color especificado.

text-shadow

Agrega una sombra del color especificado al texto.

text-decoration-color

Cambia el color de la decoración apdivcada al texto.
